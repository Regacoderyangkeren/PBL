# Project Management App

A project management application built with **Android (Kotlin + Jetpack Compose)** and a **custom Laravel REST API**.  
The app supports authentication, project & task management, offline caching, and secure API communication.

This project is developed as an educational / portfolio project with a focus on **clean architecture, security, and scalability**.

---

## Tech Stack

### Android (Frontend)
- Kotlin
- Jetpack Compose
- Material 3
- MVVM Architecture
- Navigation Compose
- Retrofit + OkHttp
- Room Database
- DataStore
- EncryptedSharedPreferences

### Backend
- Laravel (API-only)
- RESTful API
- JWT Authentication
- PostgreSQL / MySQL
- OAuth (Google & GitHub – self-hosted)
- Redis (optional)
